Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celcius and let the user know what the corresponding temperature is in Fahrenheit. 

puts "Enter degrees in Celcius:"
celcius_input = gets.chomp.to_f

def celcius_to_fahrenheit(a)
    return ((a * 1.8) + 32).round(2)
end

puts "The temperature is #{celcius_to_fahrenheit(celcius_input)} degrees Fahrenheit."

puts "Enter an integer."

input_num = gets.chomp.to_i
right_num = input_num % 10

def ordinal_value(full_value,right_value)
    if (full_value == 11) || (full_value == 12) || (full_value == 13)
        return "th"
    elsif right_value == 1
        return "st"
    elsif right_value == 2
        return "nd"
    elsif right_value == 3
        return "rd"
    else
        return "th"
    end
end

puts "That's the #{input_num.to_s + ordinal_value(input_num,right_num)} item!"

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

puts "How many items do you want to see?"

user_input = gets.chomp.to_i

user_input.times do |i|
    i += 1
    
    if (i % 3 == 0) && (i % 5 == 0)
        puts "Foobar"
    elsif i % 3 == 0
        puts "Foo"
    elsif i % 5 == 0
        puts "Bar"
    else
        puts i
    end  
end
